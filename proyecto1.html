La Zarzuela

TI-131 Programación II I Cuatrimestre 2024 Colegio Universitario de Cartago
Objetivo del proyecto

Aplicación de escritorio desarrollada para gestionar operaciones de una proveeduría distribuidora. Implementa arquitectura en capas, Programación Orientada a Objetos (POO), manejo avanzado de archivos, validación de datos robusta y control exhaustivo de errores.

Tecnologías utilizadas
IDE: Visual Studio Community 2022
Framework: .NET 7
Lenguaje: C#
Arquitectura: Capas (Presentación, Lógica, Datos)
Repositorio en GitHub
Ver código fuente en GitHub
Funcionalidades principales
  • Gestión completa de productos y inventario
  • Registro de proveedores y clientes
  • Control de compras y ventas
  • Generación de reportes y estadísticas
  • Validación de datos en tiempo real
  • Manejo de archivos para persistencia
  • Sistema de manejo de errores robusto
