Cajero Automático
con Tkinter

Tarea Programada #6 Python + SQL Server 2024
Objetivo del proyecto

Sistema completo de cajeros automáticos desarrollado con Tkinter (interfaz gráfica) y SQL Server 2016. Implementa selección de cajeros disponibles, operaciones bancarias (depósitos, retiros, consultas) con validaciones y auditoría completa de transacciones en base de datos.

Tecnologías utilizadas
Lenguaje: Python 3.x
Interfaz: Tkinter (GUI)
Base de datos: SQL Server 2016 (Nivel 130)
Tablas: Usuarios, Bancos, Cuentas, Auditoría
Repositorio fuente
Tarea #6 en GitHub
Funcionalidades principales
  • Menú principal con selección de cajeros (estado: libre/ocupado/mantenimiento)
  • Autenticación de usuarios con credenciales simuladas
  • Transacciones: depósito, retiro, consulta de saldo, cambio de cajero
  • Validaciones de montos y límites de transacción
  • Base de datos relacional con tablas de auditoría
  • Diagrama Entidad-Relación diseñado
  • Carga automática de datos iniciales
Características técnicas
Interfaz gráfica: Tkinter con estilos personalizados
Estados cajero: Libre, Ocupado, Mantenimiento, Fuera de servicio
Persistencia: SQL Server con tablas relacionales
